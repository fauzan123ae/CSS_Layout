Kesimpulan skenario 1 : Fungsi flexbox untuk mengatur wadah_flex dan elemen di dalamnya menjadi sejajar ke kanan.

Kesimpulan skenario 2 : Kita bisa menggunakan selector .wadah_flex > div untuk mengatur style pada setiap div anak dari class wadah_flex.

Kesimpulan skenario 3 : Fungsi flex-direction untuk mengatur arah urutan pada elemen-elemen yang ada dalam class wadah_flex. Column untuk vertikal, row untuk horizontal, atau bisa juga untuk lawan arahnya dengan menggunakan reverse.
